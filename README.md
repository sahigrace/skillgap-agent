🎯 SkillBridge AI: The Agentic Career Coach
🚀 Overview
SkillBridge AI is an intelligent talent acquisition and upskilling agent designed to solve the "Skill Gap" problem. Unlike traditional resume parsers that only look for keywords, SkillBridge AI uses Gemini 1.5 Flash to perform semantic analysis—understanding the logic behind a candidate's experience and how it translates to new roles.

🧠 Key Agentic Features
Semantic Alignment Matrix: Automatically extracts core competencies from both Job Descriptions and Resumes to identify "Matches," "Partial Matches," and "Critical Gaps."
Adjacent Skills Reasoning: A unique logic engine that explains how a candidate's existing background (e.g., Electronics & Communication) provides a foundation for learning software-specific tools (e.g., Spring Boot).
Automated Learning Path: Generates a high-intensity, 3-day roadmap focused specifically on the #1 priority gap found during analysis.
Interactive Live Interviewer: A real-time feedback loop where the AI poses a technical challenge and grades the user's response with a score and constructive feedback.
🛠️ Technical Architecture
Frontend: Streamlit (Custom CSS for a modern SaaS interface)
AI Engine: Google Generative AI (Gemini 1.5 Flash)
PDF Processing: PyPDF2 for raw text extraction
State Management: Streamlit Session State for maintaining interview context
